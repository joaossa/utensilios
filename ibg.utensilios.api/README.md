# ibg.utensilios.api

API Node.js + TypeScript + Express + Knex para o projeto IBG - Utensilios.

## Scripts principais

```sh
npm install
npm run db:migrate
npm run db:seed
npm run dev
```

## Variaveis de ambiente

Use `DATABASE_URL` apontando para o banco local `Utensilios` e `JWT_SECRET` para autenticacao.
